# Servidores

## para arrumar o setup do projeto
```
npm install
```

### para rodar os servidores
```
npm run udp-server
```
```
npm run tcp-server
```
### para rodar os clientes
```
npm run udp-client
```
```
npm run tcp-client
```
### Descrição.
Site feito para trabalho com servidores com sockets udp e tcp, ligue o servidor desejado e seu cliente especifico
